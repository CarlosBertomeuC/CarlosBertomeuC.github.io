Maquetación: Flexbox

Flexbox es un modelo de diseño unidimensional que permite alinear y distribuir espacio entre los elementos en un contenedor, incluso cuando su tamaño es desconocido y/o dinámico.


/* Ejemplo básico de Flexbox */
.container {
    display: flex;
    justify-content: center; /* Centra los elementos horizontalmente */
    align-items: center; /* Centra los elementos verticalmente */
    height: 100vh;
}

.item {
    background-color: #4CAF50;
    padding: 20px;
    color: white;
}

Propiedades de Texto

Las propiedades de texto permiten personalizar la apariencia del texto en una página web, incluyendo el tamaño, color, alineación, y más.


/* Propiedades de texto */
h1 {
    font-size: 2em; /* Tamaño de la fuente */
    text-align: center; /* Alineación del texto */
    color: #333; /* Color del texto */
}

p {
    line-height: 1.6; /* Altura de línea */
    text-transform: uppercase; /* Transformación del texto */
}

Modelo de Cajas (Box Model)

El modelo de cajas en CSS describe el diseño y el espacio que rodea a los elementos HTML.


/* Ejemplo del Modelo de Cajas */
.box {
    width: 300px; /* Ancho */
    padding: 20px; /* Espaciado interno */
    border: 2px solid #000; /* Borde */
    margin: 10px; /* Espaciado externo */
    background-color: #f0f0f0; /* Color de fondo */
}

Colores y Fondos

CSS permite establecer colores y fondos para elementos, utilizando valores de color, imágenes, y gradientes.


/* Colores y fondos */
body {
    background-color: #fff; /* Color de fondo */
    color: #333; /* Color del texto */
}

div {
    background-image: url('imagen.jpg'); /* Imagen de fondo */
    background-size: cover; /* Tamaño de la imagen */
    background-repeat: no-repeat; /* Sin repetición */
}

Animaciones

CSS permite crear animaciones para hacer que los elementos cambien de estilo de forma gradual.


/* Ejemplo de animación en CSS */
@keyframes mover {
    from {
        transform: translateX(0);
    }
    to {
        transform: translateX(100px);
    }
}

.elemento {
    animation: mover 2s infinite; /* Aplicación de la animación */
}

Media Queries

Las media queries permiten aplicar estilos específicos según las características del dispositivo, como el tamaño de la pantalla.


/* Media Query para pantallas pequeñas */
@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }

    .container {
        flex-direction: column; /* Cambiar a diseño vertical */
    }
}
